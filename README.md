# Gradient Descent & Perceptron Learning

এই নোটবুকে আমি স্ক্র্যাচ থেকে Gradient Descent algorithm implement করে Linear Regression model তৈরি করেছি এবং Perceptron algorithm সম্পর্কিত কিছু গুরুত্বপূর্ণ conceptual প্রশ্নের উত্তর দিয়েছি।

## Implementation:
- sklearn থেকে regression dataset তৈরি করেছি
- নিজে থেকে GDRegressor class বানিয়েছি
- Gradient Descent ব্যবহার করে optimal parameters (slope m এবং intercept b) খুঁজে বের করেছি
- প্রতিটা iteration এ কীভাবে loss কমছে এবং parameters converge করছে সেটা observe করেছি

## Conceptual Questions:
- Perceptron এ −1/+1 class label কেন ব্যবহার করা হয়?
- Gradient Descent সাধারণত gradient subtract করে, কিন্তু Perceptron কেন add করে?
- Perceptron Learning Rule কীভাবে Gradient Descent এর একটি রূপ?
- Perceptron Loss Function এর সীমাবদ্ধতা কী?
